Section IV Lifecycle Manager : Lifecycle Manager Components : Change Passwords
 
Change Passwords
Use the Change Passwords tab to issue requests to auto-generate or manually set account passwords for selected identities. Click Change Passwords then select For Me to manage your passwords or select For Others to mange passwords for others. To open the Select Identities page, select For Others.
Select an identity from the Available Identities table to display the password request actions and a table that includes all of the selected identity’s account applications.
 
Table 2—Change Passwords Page
Column Name
Description
Password Request Actions:
Set passwords for the selected accounts
Select this option to request a new password. Type the individual passwords for each account in the New Password and Confirm Password fields.

Choose Synchronize passwords for selected accounts to use the same password for all selected accounts.
Generate passwords for the accounts
Select this option to request the auto-generation of passwords based on established password policies for all selected accounts.

Choose Generate a single password to synchronize for the selected accounts to use the same password for all selected accounts.
Accounts:
Account ID
Name of the account.
Application
The application specific to the Account ID.
Status
The current status of the account.
Last Refresh
The date the account information in IdentityIQ was last updated.
Click Submit after all selections are completed to display the Review and Submit page.
