Section III Using IdentityIQ : Reports : Role Management Reports : Role Change History Report : Role Properties
 
Role Properties
The following criteria determines what information is included in this report. You can use any combination of options to build a report.
Note: Selecting NO options from a list indicates that ALL options in the list are included in the report.
 
Table 5—Role Change History Report Properties
Option
Description
Change Start and End Date(s)
Filter request based on request date:
Start Date — all changes made on or after the selected date.
End Date — all changes made on or before the selected date.
Role Status
Include only active roles or only inactive roles in the report.
Type
Select types of roles to include in the report.
Applications
Select the applications to include in the report. If no applications are specified, all applications are included.
Click the arrow to the right of the suggestion field to display a list of all applications, or enter a few letters in the field to display a list of applications that start with that letter string.
Only roles associated with the selected applications are included in this report.
Owners
The list of role owners to include in this report. If no role owners are specified, the roles for all owners are included.
Click the arrow to the right of the suggestion field to display a list of all role owners, or enter a few letters in the field to display a list of role owners that start with that letter string.
