Section III Using IdentityIQ : Advanced Analytics : Activity Search : Activity Search Results : Result Options
 
Result Options
Use the drop-down list to save search criteria to use in future searches:
• Save or Update Search — save the search for your own use. A list of saved searches displays at the top of the search page every time you log in.
• Save or Update Search as Report — searches saved as reports are added to your list of reports and can be scheduled to run on a continuous basis. See Reports.
