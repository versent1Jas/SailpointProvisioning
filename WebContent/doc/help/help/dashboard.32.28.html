Section III Using IdentityIQ : IdentityIQ Dashboard : View Work Item Page : Summary
 
Summary
The summary section contains the following:
 
Field Name
Description
Requestor
The name of the IdentityIQ user that assigned this work item to you.
Description
A brief description of the work item.
Date
The date the work item was assigned.
Expiration
The date by which the work item must be completed, if applicable.
Severity
Severity of the work item.
History
The history of this work item, including previous owner history.
